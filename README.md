# Alfabetiza

Alfabetiza is a developer-maintained Next.js application for tracking schools, classes, students, reading levels, reading history, teacher sessions, and administrative audit data.

The project uses Next.js 16, React 19, TypeScript, Prisma, PostgreSQL, next-intl, Vitest, and Playwright.

## Developer Documentation

The full developer handbook lives in [`docs/`](docs/README.md).

- English: [`docs/en/01-getting-started.md`](docs/en/01-getting-started.md)
- Português do Brasil: [`docs/pt-BR/01-primeiros-passos.md`](docs/pt-BR/01-primeiros-passos.md)

## Common Commands

```bash
pnpm install
pnpm dev
pnpm test
pnpm run test:e2e
```

Before changing Next.js code, read the relevant local guide in `node_modules/next/dist/docs/` and follow `AGENTS.md`.
